# Alma H-1B Lottery Registration

Alma is an AI-native immigration law firm. Every spring, its attorneys file H-1B
lottery registrations for their clients through the USCIS portal, often hundreds
at a time, against a window that opens for just two weeks once a year. This
project digs into how to take that off their plate: where the real work actually
hides, a PRD for automating it, and a clickable prototype of the attorney-facing
tool that does it.

Live prototype: https://h1b-console.vercel.app
